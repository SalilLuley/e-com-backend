## Description

This is an e-commerce skeleton repository made with clean code Architecture using Nest.js Framework. It used my sql as a backend.

It includes the following modules

1. User Auth using Passport
2. Bcrypt for password hashing
3. User Address
4. User Payment
5. Cart Item
6. Payment Details
7. Product
8. Product Category
9. Product Inventory
10. Discount
11. Order Details
12. Order Items

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
